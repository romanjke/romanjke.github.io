Сайт расписания занятий и экзаменов ИВГПУ

Ссылка:
http://rasp.ivgpu.com
Описание:
Сайт расписания занятий и экзаменов ИВГПУ, на котором пользователь с помощью формы, размещенной на главной странице, может выбрать определенную учебную группу / преподавателя и затем найти соответствующее расписание занятий / экзаменов как на две недели, так и на определенный день. Полученное расписание можно скачать в формате .xlsx.
Пользователи, наделенные специальными правами, могут добавлять, изменять, удалять информацию о занятиях / экзаменах, а также некоторые другие сведения.
Платформа:
Laravel
Моя роль:
Полная разработка сайта.
Проектирование структуры сайта, проектирование архитектуры базы данных, установка и настройка фреймворка, добавление необходимых пакетов через Composer, программирование функционала с помощью API фреймворка, наполнение первоначальным контентом, создание поддомена, размещение на хостинге, дальнейшая поддержка.

Сайт диссертационных советов ИВГПУ

Ссылка:
http://ds.ivgpu.com
Описание:
Сайт диссертационных советов ИВГПУ, на котором размещается информация о диссертациях и диссертационных советах. Каждая диссертация имеет свой статус: «Представленные к защите», «Защищенные» или «Добавленные в архив».
Пользователи, наделенные специальными правами, могут добавлять, изменять и удалять информацию о диссертациях / диссертационных советах.
Платформа:
Laravel
Моя роль:
Полная разработка сайта.
Проектирование структуры сайта, проектирование архитектуры базы данных, установка и настройка фреймворка, добавление необходимых пакетов через Composer, программирование функционала с помощью API фреймворка, наполнение первоначальным контентом, создание поддомена, размещение на хостинге, дальнейшая поддержка.
